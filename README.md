# 🎮 Tic-Tac-Toe Game

A simple browser-based Tic-Tac-Toe game built with **HTML, CSS, and JavaScript**.  
Play as Player X or Player O, and enjoy a classic game of strategy and fun.

- Reset & New Game buttons
- Responsive and interactive UI
